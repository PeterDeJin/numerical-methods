# 數值方法 — 課程作業

數值方法／數值分析課程作業，每題皆自行實作演算法並以課本範例驗證正確性。

## 內容
| 程式 | 主題 |
|------|------|
| `HW1.ipynb` | SOR（逐次超鬆弛）疊代法解線性方程組 $Ax=b$ |
| `HＷ2.ipynb` | 共軛梯度法 (Conjugate Gradient) 解線性方程組 |
| `HW3.ipynb` | 冪次法 (Power Method) 求主特徵值 |
| `HW4.ipynb` | Crank-Nicolson 法解偏微分方程（熱方程）|

對應的書面作業：`數值作業HW1–HW4.docx`；課程大綱：`系上數值課程大綱.docx`。
